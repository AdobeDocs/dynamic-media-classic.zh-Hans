# 导出源和编码视频{#exporting-source-and-encoded-videos}

在浏览面板中，源视频标为“主视频”。编码视频标为“视频”。您可以导出源视频和主视频。要导出视频文件，请在浏览面板中将其选中，并选择“文件”>“导出”。

请参阅[从Dynamic Media Classic](exporting-assets-from-dmc.md#exporting-assets-from-dmc)导出资源。
